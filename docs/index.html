Not: Bu sayfa projenin statik tanıtımıdır. Canlı sürüm için yukarıdaki Vercel butonunu kullan veya aşağıdaki adımlarla yerelde çalıştır.

Yerelde Çalıştır

Node.js 20+ yeterli — Supabase anahtarı olmadan DEMO modunda açılır

# depoyu klonla
git clone https://github.com/R0YC0LD/RapLab.git
cd RapLab

# bağımlılıkları kur ve başlat
npm install
npm run dev

# http://localhost:3000 → "DEMO MODU" bandıyla açılır

Canlıya Alma

Gerçek kayıt/giriş ve veritabanı için — ayrıntılar repodaki KURULUM.md dosyasında

Supabase projesi aç (ücretsiz) ve Project URL, anon, service_role anahtarlarını kopyala.

Vercel'e aktar: yukarıdaki "Vercel'de Canlıya Al" butonuna tıkla, üç anahtarı Environment Variables bölümüne gir.

Şemayı yükle: npx supabase db push — tablolar, trigger'lar ve RLS politikaları otomatik kurulur.

Yayında: DEMO bandı kaybolur; kayıt, giriş, sanatçı başvurusu ve paneller gerçek veriyle çalışır.
